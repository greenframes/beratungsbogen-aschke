# Beratungsbogen PWA (Native Layout, ohne Overlay)

## Start (Lokal)
1) Entzippen
2) Terminal im Ordner
3) `python -m http.server 8080`
4) iPad: `http://<IP-DEINES-LAPTOPS>:8080`
5) Safari -> Teilen -> „Zum Home-Bildschirm“ (optional)

## Export
Button „Bestätigen & speichern“
- Rückfrage: „Sind alle Angaben korrekt?“
- Danach iOS Druckansicht -> Teilen -> „In Dateien sichern“ (PDF)
- Popup „Vielen Dank!“ -> OK -> Formular wird geleert

## Dateiname
Die App setzt den Dokumenttitel auf:
`Nachname_Vorname_YYYY-MM-DD`
iOS übernimmt diesen Namen oft als Vorschlag (nicht garantiert).

## Hinweis: „Direktlink zum Speicherort + Passwort“
iPadOS erlaubt Webapps keinen direkten Zugriff auf den Dateien-Speicherort.
Wenn du einen „geschützten Admin-Bereich“ möchtest (PIN) um lokale Entwürfe/Exports in der App zu sehen, sag Bescheid – das können wir ergänzen.
